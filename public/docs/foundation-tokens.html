Border radius
Six stops from sharp to pill. Larger radii signal interactivity and containment — smaller radii signal precision and data.
radius.none 0px Dividers, bars
radius.sm 4px Badges, tags
radius.md 8px Inputs, buttons
radius.lg 12px Cards, dropdowns
radius.xl 16px Panels, modals
radius.2xl 20px Hero, feature cards
radius.full 9999px Pills, toggles
03
Elevation
Light mode uses box-shadows. Dark mode uses background-lightening — shadows are invisible on dark surfaces.
Light mode — box-shadow
shadow.none Flat — page base
shadow.xs Inputs on focus
shadow.sm Cards, dropdowns
shadow.md Raised cards, popovers
shadow.lg Modals, drawers
Focus rings
focus.steel Inputs · light mode
focus.steel-dark Inputs · dark mode
focus.gold Primary actions
focus.red Error states
Dark mode — surface lightening

elevation.surface #1C2D39 Cards, inputs
elevation.raised #1A2830 Stat cells
elevation.modal #243444 Modals, drawers
elevation.overlay #2C3E50 Command palette
04
Motion
Two axes: duration (how long) and easing (how it moves). Use the minimum duration that still feels intentional.
Duration
Scale relative to max of 400ms. Prefer fast and default for UI; reserve slow+ for entrances.
duration.instant
75ms
Checkbox tick, toggle snap
duration.fast
100ms
Hover states, badge swap, tooltip show
duration.default
200ms
Most transitions — color, border, opacity
duration.slow
300ms
Modals, panels, drawers entering
duration.slower
400ms
Staggered reveals, onboarding flows
Easing
Ease-out is the default — elements decelerate into position. Ease-in for exits. Spring for delight moments.
easing.ease-out
cubic-bezier(0, 0, 0.2, 1)
Entering elements — default
easing.ease-in
cubic-bezier(0.4, 0, 1, 1)
Exiting elements — dismiss
easing.ease-in-out
cubic-bezier(0.4, 0, 0.2, 1)
Elements moving position
easing.linear
linear
Spinners, progress bars
easing.spring
cubic-bezier(0.34, 1.56, 0.64, 1)
Toasts, confirmations, delight
